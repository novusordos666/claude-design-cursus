## STAP 1 — Waarom geen generieke "AI vibe"

Iedereen maakt een "AI vibe" hero. Gekleurde blob. Particles. Generic. Klant vergeet 't meteen.

**Wij doen iets anders:** we tonen letterlijk onze business.

**187N Agent Command Scene:**
- Het 187N infinity logo in het midden als command
- 4-6 agent silhouettes orbiting
- Data-beams connecting
- Holografische grid floor

Je klant ziet niet alleen je logo — je klant ziet je hele bedrijfsmodel. AI agents rond een central system, 24/7, altijd aan. **Dit is agency-level branding.**

## STAP 3 — Beat 2: Nano Banana Pro (stilstaande image)

Open **Higgsfield.ai → Nano Banana Pro**.

**Plak de 187N Agent Command Scene prompt uit je dashboard** (rechts, kopieer-button).

**Key regels die in de prompt staan:**
1. LEFT 40% negative space voor headline overlay
2. Centerpiece: 3D infinity-loop front-facing, glossy obsidian, emerald rim-light
3. Orbital layer: 4-6 humanoid wireframe silhouettes — faceless, agent-achtig
4. Data-beams van elke agent naar de loop's chambers
5. Grid floor met perspective, receding naar vanishing point
6. Emerald particles, scan-lines, void black domineert
7. Kleuren STRICT: #050a08 + #00ff88. Geen blauw, paars, cyan.
8. No text, no baked logos — het logo in de scene IS het brandmark

**Wacht 20-40 sec voor generatie.** Download als `hero-still.png`.

## STAP 5 — Beat 4: Seedance 2.0 Animatie

Open **Higgsfield.ai → Seedance 2.0**.

**Upload `hero-still.png` als starting frame.**

**Plak de Seedance motion-prompt uit je dashboard** (Concept A: "loop rotation + data flow").

**KRITIEKE SETTINGS — check dit dubbel:**

| Setting | Waarde |
|---------|--------|
| Aspect ratio | 16:9 |
| Resolutie | 1080p |
| Length | 8 seconden |
| **Enhance Prompt** | **UIT** — kritiek, anders krijg je iets anders |

**Wat de prompt zegt:**
- GEEN 360 rotation. Logo blijft front-facing.
- Camera z-axis only — eerst 4s dolly back, laatste 4s push-in.
- Logo breathing pulse 1.0 tot 1.03 op 2-sec cycle.
- Agent silhouettes subtle motion (één knikt, één tilt, één activate).
- Data-beams pulse 2-3 per sec staggered.
- Activity ring elke 2-2.5s vanuit loop-center.
- Scan-lines subtle continuous + bright sweep elke 3-4s.

**Wacht 2-5 min.** Download als `hero-motion.mp4`.

> **ACTIE:** Upload still, plak motion-prompt, check settings (Enhance Prompt UIT!), genereer. Typ `1` als mp4 op disk staat.

## STAP 6 — Beat 5: MP4 Swap + Mobile Fallback

Terug naar Claude Design — **zelfde chat als Beat 3**.

**Sleep `hero-motion.mp4` direct in chat. Plak:**

```
Can we swap the still image for the video I just uploaded?
Keep everything else identical (layout, text, CTAs, colors).
Video should autoplay, muted, loop seamlessly.
On mobile: fall back to the original still image for data savings.
```

Claude Design begrijpt het. Vervangt de achtergrond. Zelfde layout, zelfde tekst, zelfde CTAs. Maar nu **cinematic video** achter.

"Autoplay muted loop" gebeurt automatisch.

**Mobile fallback check:** toggle mobile preview in Claude Design. Mobile users zien stilstaande image (data-besparing). Niemand wil 10MB videoload op 4G. Claude Design regelt dit — video voor desktop, still voor mobile.
