    Create a cinematic hero image for 187N.AI, enterprise AI infrastructure agency.

COMPOSITION: Split layout - LEFT 40% negative space for headline overlay (MUST be empty), RIGHT 60% the scene.

SCENE (right 60%):
- CENTERPIECE: 3D infinity-loop front-facing, glossy obsidian with emerald rim-light glow, levitating
- ORBITAL LAYER: 4-6 humanoid wireframe silhouettes (faceless, agent-like) orbiting the loop
- DATA-BEAMS: thin emerald light-streams from each agent to the loop's chambers
- FLOOR: holographic grid with perspective, receding to vanishing point
- ATMOSPHERE: subtle emerald particles, horizontal scan-lines across scene, void black domination

COLORS STRICT: #050a08 (void black dominant) + #00ff88 (emerald accents with glow). NO blue, purple, cyan, pink.

NO text, no logos baked in - the infinity-loop IS the brandmark.

Style: cinematic key-lit, cybersecurity ops-center at night, film-grade lighting, 16:9 aspect ratio.
    Seedance 2.0 motion prompt - 187N Agent Command Scene (loop rotation + data flow).

STARTING FRAME: uploaded still image (hero-still.png).

DURATION: 8 seconds.
ASPECT: 16:9.
RESOLUTION: 1080p.
ENHANCE PROMPT: OFF (critical - otherwise output diverges from prompt).

CAMERA:
- Z-axis ONLY. NO rotation. NO 360 spin.
- First 4s: slow dolly back (zoom out ~10%)
- Last 4s: smooth push-in (zoom in ~10%)

CENTERPIECE - infinity loop:
- Front-facing throughout (no flip, no rotation)
- Breathing pulse scale 1.0 -> 1.03 on 2-second cycle

AGENT SILHOUETTES (orbital layer):
- Subtle motion: one nods, one tilts, one activates (brief glow flare), one swaps position slightly
- NOT a coordinated dance - organic, idle-ops feel

DATA-BEAMS:
- Pulse 2-3 per second, staggered between agents
- Emerald color intensifies at mid-beam then fades

ACTIVITY RING:
- Expanding ring emission from loop-center every 2-2.5 seconds
- Fades out at 40% of scene width

SCAN-LINES:
- Subtle continuous horizontal drift
- Bright sweep line every 3-4 seconds

Overall mood: control room at night, always-on, quiet confidence.
    In Claude Design: Export dropdown -> "Handoff to Claude Code" -> copy. This generates a ~hundreds-of-lines prompt with exact colors, fonts, CSS classes, video URLs, responsive breakpoints, section structure.

Then in Claude Code terminal:
/build-page

Paste the export. Claude Code scaffolds Vite + React + TypeScript, installs Tailwind + lucide-react, creates components/sections for each section, builds liquid-glass CSS class with conic border-mask, downloads texture.png placeholder.

Then:
/deploy-site

Runs: git init, gh repo create, vercel link, vercel deploy --prod. Output: orbis-nft.vercel.app (or your variant).
